Welcome to Industrial Blog. This is a demonstration of a high-performance, reading-focused blog platform built with Next.js, MDX, and Tailwind CSS.

## Why Minimalism Matters

Minimalism in web design isn't about removing features—it's about removing distractions. Every element on this page serves a purpose:

1. **Typography** - Clean, readable fonts that don't distract
2. **Whitespace** - Generous margins that give content room to breathe
3. **Color** - A restrained palette that won't overwhelm

## The Stack

This blog uses:

- **Next.js 14** with App Router for routing and rendering
- **MDX** for content authoring with component support
- **Tailwind CSS** for styling
- **Shadcn/UI** for accessible components
- **Fuse.js** for search

## Getting Started

To add a new post, simply add a new `.mdx` file to the `content/posts` directory:

```mdx
---
title: Your Post Title
excerpt: A brief description of your post.
date: 2024-01-02
category: Category
---

Your content here...
```

That's it. The build system will automatically pick it up and generate the pages.
